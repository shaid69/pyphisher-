## Installation

### Requirements
- Python 3.6+
- PHP
- SSH
- cURL

### Linux/macOS
```bash
git clone https://github.com/shaid69/pyphisher-.git
cd pyphisher-
pip3 install -r requirements.txt
chmod +x pyphisher.py
```
